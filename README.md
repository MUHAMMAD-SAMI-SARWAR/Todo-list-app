# 📝 Task Manager App

A sleek, functional, and responsive **To-Do List Application** built with vanilla JavaScript. This app helps users stay organized by managing daily tasks with features like filtering, data persistence, and dark mode.

---

## ✨ Features

* **Add & Edit Tasks:** Easily create new tasks or update existing ones without losing context.
* **Task Persistence:** Uses `localStorage` so your tasks remain saved even after refreshing the browser or closing the tab.
* **Status Filters:** Filter and view tasks easily based on their current status: **All**, **Active**, or **Completed**.
* **Dark Mode Toggle:** Smooth transition between Light and Dark themes for a better visual experience.
* **Live Task Counter:** Real-time tracking of how many tasks are left to complete.
* **Keyboard Support:** Press `Enter` inside the input box to add tasks quickly.
* **Prevent Duplicates:** Automatically checks and prevents adding identical tasks.
